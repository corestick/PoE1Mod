# PoE1Mod
Pillars of Eternity 1 Mod

UMM 필수 : https://www.nexusmods.com/site/mods/21

모드 목록

AchievementCheck
콘솔 활성화 실행 시 업적 활성화 안되게 막아줌

AdjustBuffDuration
버프 지속시간을 지정한 시간으로 계산. buff_whitelist.json 파일 whitelist 에 등록된 스킬만 적용

AdjustDifficulty
적의 체력, 스탯을 조정

AdjustMovementSpeed
이동속도를 조정. 일반/은신 상태 각각

CampingSuppliesMax
캠핑 보급품 한도 99로

CharacterStatsEditor
선택된 캐릭터의 스탯, 스킬 수정가능
게임 중 umm 설정창 (Ctrl + F10) 열어서 수정
스킬은 수정사항이 세이브 게임에 저장되지 않고 모드폴더에 저장되므로, 모드삭제 시 수정사항이 날아가므로 주의

CipherFocus
영매사의 집중 초기값을 수정

DisableFriendlyFireAOE
범위 마법이 아군에게 피해를 주지 않음

DoubleTalentPoints
레벨업 시 얻는 재능 포인트를 수정

EnableCombatOnlyAbility
전투 전용 마법을 비전투 시에도 사용 가능하게

EnablePerEncounterSpells
사용횟수가 휴식 시에 충전되는 능력을 전투종료 시 충전 (주문 포함)

GrimoireSlots
마법사 마법책 슬롯을 4개에서 7개로 늘림

InfiniteSpellCasts
무한 시전 모드

RemoveAbilityPointsCap
만렙(16?) 후 스킬 포인트 획득 테이블 1렙부터 다시 시작 되게

RemovePlayerLevelCap
만렙을 99로 늘림

StackableMod
장비/스킬의 동일 능력치 상승 중복 허용

StrongholdAdventureTurn
본거지 이벤트 발생 빈도 (턴) 수정

TrapDetectAlways
은신 모드 없이 함정 찾기 가능
